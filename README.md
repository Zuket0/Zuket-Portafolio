# Portafolio Profesional - Juan Chavez E.

Bienvenido al repositorio de mi portafolio personal. Soy desarrollador Full Stack y estudiante de Ingeniería de Sistemas enfocado en diseñar arquitecturas de software robustas, optimizar procesos empresariales y crear interfaces web eficientes.

## Sobre el Proyecto

Este sitio web es una aplicación estática interactiva diseñada para presentar mi experiencia técnica y mis desarrollos más recientes. Cuenta con animaciones fluidas, un carrusel interactivo desarrollado en JavaScript puro y modales descriptivos estilo "caso de estudio" para cada proyecto.

## Arquitectura y Tecnologías

El portafolio fue construido priorizando el rendimiento, la accesibilidad y un diseño responsivo sin depender de frameworks pesados de JavaScript:

* **Estructura y Estilos:** HTML5, CSS3 (Variables CSS, Flexbox, Grid)
* **Interactividad:** JavaScript (Vanilla JS, Fetch API)
* **Diseño UI:** Bootstrap 5.3
* **Iconografía:** FontAwesome 6
* **Despliegue:** Vercel

## Proyectos Destacados Incluidos

El portafolio detalla soluciones arquitectónicas de diferentes sectores:

* **Zorval Web App:** Plataforma web para control de flujo de efectivo.
* **COSTOS MP:** Software corporativo en Java y SQL Server para cálculo de materias primas.
* **Lunix TV:** Plataforma de streaming de video para Android TV.
* **Habil Metal Import:** Catálogo web dinámico B2B desarrollado en PHP.
* **iBusiness MTM & IAAUTOS:** Plataformas corporativas en Python para reportes técnicos e inspecciones vehiculares.
