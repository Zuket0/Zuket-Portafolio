Optimización COSTOS MP

Software corporativo desarrollado en Java para la centralización y cálculo inteligente de costos de materias primas. Su núcleo es un algoritmo de priorización que asegura la exactitud de los datos financieros en entornos de alta demanda.

  • Integración de Sistemas: Sincronización de datos bidireccional entre SQL Server, archivos Excel y sistemas internos de la empresa.
  • Cálculo Dinámico: Algoritmos de priorización de precios para determinar costos reales de producción.
  • Respaldo Automático: Lógica de detección de vacíos (fallback) para mantener la operación ininterrumpida si faltan datos en la base principal.
Lunix TV Streaming

Plataforma de streaming de video híbrida construida para operar de manera nativa tanto en dispositivos móviles Android como en televisores inteligentes (Android TV), brindando una experiencia de consumo multimedia de alto rendimiento.

  • Interfaz Adaptativa: Implementación de interfaz dual (Touch para móviles y Leanback UI para navegación con control remoto en TV).
  • Optimización de Recursos: Sistema avanzado de manejo de caché de imágenes y carga asíncrona para evitar saturación de memoria.
  • Arquitectura JSON: Consumo de APIs REST para alimentar dinámicamente el catálogo de contenidos.

Formulario de Mantenimientos

Plataforma web corporativa con backend en Python diseñada para automatizar y estandarizar la generación de reportes de mantenimiento técnico en campo.

  • Autenticación Segura: Sistema de login integrado con SQLite manejando cifrado Hash para proteger los datos corporativos.

Inspección Técnica de Autos

Sistema corporativo integral diseñado para digitalizar y agilizar las inspecciones vehiculares dentro de concesionarios automotrices.

  • Búsqueda Avanzada: Motor de filtrado indexado por VIN (Número de Identificación Vehicular) y paginación para bases de datos a gran escala.
  • Reportes de Cumplimiento: Exportación dinámica mediante pdfkit para certificar las inspecciones ante la gerencia.
